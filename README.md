# Portfolio — Sanyukta Muley

Static personal portfolio site: HTML + CSS only.

## Run locally

Open `index.html` in a browser, or serve the folder (needed if you add assets that block `file://`):

```bash
cd portfolio
python3 -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080).

## Deploy

Upload the `portfolio` folder to [GitHub Pages](https://pages.github.com/), [Netlify](https://www.netlify.com/), or [Vercel](https://vercel.com/) as a static site. Set the publish directory to the repository root (or this folder if the repo only contains the site).
